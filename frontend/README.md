# FRONTEND

## DESCRIPTION

Responsible for providing an intuitive and responsive user interface that allows users to view weather information based on their location. It communicates with backend services through the API Gateway and also integrates with PokéAPI and SWAPI to display additional data fetched from external sources. The frontend presents real-time updates using modern UI patterns and best practices.
